## 1?? **start.bat** - Universal Launcher (HAUPT-LAUNCHER)

### **Verwendung**:
```cmd
# Einfach doppelklicken
start.bat
```

### **Was macht er?**:
- Zeigt Menü mit allen Versionen
- Du wählst die Version (1-5)
- Startet die gewählte Version

### **Verfügbare Versionen**:
```
1. Modern UI        ? NEU! Modernes Design
2. Guided Version   ? Geführt, Schritt-für-Schritt
3. Simple Version   ? Minimalistisch, einfach
4. Full Version     ? Alle Features, komplex
5. Test Suite       ? Für Entwickler
```

### **Features**:
```
? Automatische Browser-Erkennung (Chrome, Edge, Firefox)
? Fallback auf Standard-Browser
? Fehlerbehandlung (Datei nicht gefunden)
? Übersichtliches Menü
? Zurück zum Menü nach Fehler
```

## ?? BROWSER-ERKENNUNG

Die Launcher suchen automatisch nach Browsern in dieser Reihenfolge:

- Du kannst HTML-Datei manuell öffnen

## ?? TROUBLESHOOTING

### **Problem: "Datei nicht gefunden"**
```
Lösung:
1. Prüfe ob HTML-Datei existiert
2. Starte Launcher aus richtigem Ordner
3. Pfad: C:\Users\reid1\Documents\Spanish\
```

### **Problem: "Kein Browser gefunden"**
```
Lösung:
1. Installiere Chrome, Edge oder Firefox
2. Oder: Öffne HTML-Datei manuell
3. Rechtsklick ? Öffnen mit ? Browser wählen
```

### **Problem: "App startet nicht"**
```
Lösung:
1. Prüfe ob alle Dependencies geladen sind
2. Öffne Browser-Konsole (F12)
3. Schaue nach Fehler-Meldungen
```
