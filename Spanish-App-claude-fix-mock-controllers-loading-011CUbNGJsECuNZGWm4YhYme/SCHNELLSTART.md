## ? PROBLEME?

### App lädt nicht:
```
1. Browser aktualisieren (F5)
2. Andere Browser probieren
3. start.bat neu starten
```

### Keine Vokabeln:
```
1. Prüfen ob data/items.json existiert
2. Internet-Verbindung prüfen (erste Nutzung)
3. Browser-Cache leeren
```

### Fehler-Meldung:
```
1. Seite neu laden (F5)
2. Browser neu starten
3. start.bat neu starten
```
